Level of ISU's Quad

Objective

The objective of this project was to create a level design using the irregular paths of the Quad layout at Illinois State University. The project involved designing a level that incorporated these paths in an isometric perspective while also incorporating interesting paths and choices a player could make.


Process

The process started with a reference map of the Quad that was used to draw the layout in an isometric perspective. Once that was complete I began designing the level and architecture before ultimately cleaning up the linework for the final version.
