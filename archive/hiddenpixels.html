Objective

The goal of this project was to develop a game that was fully playable within Adobe InDesign. The game is played by trying to find the hidden "key" that allows you to travel into the next biome.


Process

My role in this project was to create the artwork for the menu, levels, and poster design. We chose to use a pixel art style and I created different themed biomes where the player would have to guess which object was the key to escape to the next world.
